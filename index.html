2Types of Recommendation Systems

üîç Three Main Approaches to Recommendations

There are three primary ways to build recommendation systems, each with its own strengths and use cases.

üë• Collaborative Filtering

The Idea: "People like you also liked..."

Recommendations based on what similar users liked or what similar items were rated highly.

Example: If you and I both loved Marvel movies, and I also loved Star Wars, the system might recommend Star Wars to you!

üìã Content-Based Filtering

The Idea: "More of what you like..."

Recommendations based on the features/characteristics of items you've liked before.

Example: If you love action movies with superheroes, the system recommends more action superhero movies!

ü§ù Hybrid Systems

The Idea: "Best of both worlds..."

Combines collaborative and content-based approaches for more accurate recommendations.

Example: Netflix uses both your viewing history AND what similar users watch to recommend content!

üéØ Focus of This Course

We'll dive deep into Collaborative Filtering - the most widely used approach in industry. We'll explore algorithms like SVD, KNN, and EASE using real movie rating data!

8Cold Start Problem & Solutions

What happens when a completely new user joins your platform? They have no ratings, no history, no similar users to compare with. This is the "cold start" problem!

Solution: Recommend popular items

üÜï New Item

Solution: Content-based features

üì± New System

Fresh platform, minimal data

Popular Items Strategy
Python 
def get_popular_items(ratings_df, min_ratings=50):
    """Get most popular items for cold start users"""
    item_stats = ratings_df.groupby('item_id').agg({
        'rating': ['count', 'mean']
    })
    
    # Filter items with sufficient ratings
    popular_items = item_stats[
        item_stats[('rating', 'count')] >= min_ratings
    ]
    
    # Sort by average rating
    popular_items = popular_items.sort_values(
        ('rating', 'mean'), ascending=False
    )
    
    return popular_items.head(10)

# For new users, recommend top popular movies
cold_start_recs = get_popular_items(ratings_df)
print("Top recommendations for new users:")
print(cold_start_recs)

9EASE Model - Embarrassingly Simple Yet Powerful

üöÄ EASE: When Simple Beats Complex

EASE (Embarrassingly Shallow Autoencoders) proves that sometimes the simplest solutions work best. It uses a closed-form mathematical solution to learn item-item similarities directly!

üßÆ How EASE Works in 4 Steps

Step 1: Compute Gram Matrix (X^T √ó X) - Item co-occurrence
Step 2: Add regularization to diagonal (prevent overfitting)
Step 3: Compute matrix inverse (closed-form solution!)
Step 4: Zero diagonal (items don't recommend themselves)

# Complete MovieLens Recommendation System
import pandas as pd
import numpy as np
from surprise import Dataset, SVD, accuracy
from surprise.model_selection import train_test_split

# 1. Load and prepare data
data = Dataset.load_builtin('ml-100k')
trainset, testset = train_test_split(data, test_size=0.25)

# 2. Train SVD model
model = SVD(n_factors=50, random_state=42)
model.fit(trainset)

# 3. Make predictions
predictions = model.test(testset)
rmse = accuracy.rmse(predictions)

# 4. Get recommendations for a user
def get_recommendations(model, user_id, n_recs=5):
    # Get all items not rated by user
    all_items = set([trainset.to_raw_iid(i) for i in range(trainset.n_items)])
    user_items = set([item for item, rating in trainset.ur[trainset.to_inner_uid(user_id)]])
    unrated_items = all_items - user_items
    
    # Predict ratings for unrated items
    predictions = []
    for item in unrated_items:
        pred = model.predict(user_id, item)
        predictions.append((item, pred.est))
    
    # Sort by predicted rating and return top N
    predictions.sort(key=lambda x: x[1], reverse=True)
    return predictions[:n_recs]

# 5. Get recommendations for user
